### Decisões Chave
- Na reunião com os arquitetos massimos, será solicitado um exemplo do material do projeto de móveis que eles entregam, para avaliação da sua completude. - Rationale: Para verificar se o nível de detalhe é suficiente para qualquer marceneiro executar o trabalho sem necessidade de complementações.

### Minutas Detalhadas
[00:01-02:00] **Preparação para a reunião sobre a falta de detalhamento de engenharia no projeto de móveis.**
- toni expressa receio para a reunião do dia seguinte com os arquitetos massimos, querendo resolver uma questão pendente para evitar problemas futuros.
- O principal problema é a falta da "parte de engenharia" no projeto de móveis. massimos (Marcos e Fer) afirmam que o marceneiro Lucena deve realizar esta etapa, enquanto Lucena alega que outros arquitetos entregam o projeto completo.
- Os marceneiros indicados pelos massimos parecem ser mais caros, conforme orçamentos feitos para gui.
- A intenção é questionar na reunião o que foi exatamente contratado, pois o entendimento era de um projeto completo.
- Caso os massimos não realizem essa parte do projeto, a proposta é que eles indiquem alguém que o faça, para que o projeto chegue completo à marcenaria e evite jogos de culpa.
- toni considera que não se deve limitar as opções de marcenaria devido a essa lacuna no projeto.
- É levantada a hipótese extrema de contratar um novo arquiteto do zero, mesmo que isso signifique perder o investimento já feito (R$ 40 mil), se isso resultar em uma economia maior (R$ 300-400 mil) no custo final da marcenaria.
- O objetivo final é ter um projeto que qualquer marceneiro (incluindo Lucena) possa executar sem necessidade de complementação.

[02:03-03:36] **Estratégia de abordagem para a reunião com os arquitetos.**
- carol sugere uma abordagem para a conversa, focando em como os massimos imaginam o trabalho com o marceneiro Lucena, dado que há uma grande possibilidade de contratá-lo devido a uma negociação envolvendo o apartamento dos interlocutores.
- A sugestão é questionar até onde vai o serviço contratado, sem afirmar que os arquitetos "não sabem" fazer, mas sim se o detalhamento de engenharia está ou não contemplado no escopo.
- Deve-se deixar claro que, se o serviço não estiver incluído, uma decisão precisará ser tomada:
    - Gastar mais com outra marcenaria.
    - Contratar um arquiteto secundário para fazer apenas os detalhamentos, questionando se massimos estariam de acordo com essa solução.
- A comunicação deve ser feita mencionando uma "grande possibilidade" de trabalhar com Lucena, e não uma certeza, para manter a negociação aberta.

[03:36-05:07] **Definição da postura e solicitação de exemplos de projetos.**
- toni discorda da ideia de pedir permissão para contratar um segundo arquiteto, defendendo uma postura mais firme: se os massimos não fazem o detalhamento, eles devem indicar quem o faça, com os custos já definidos.
- A ideia é exigir que a solução seja parte de um pacote, para que se tenha clareza do custo total para obter um projeto que qualquer marceneiro (incluindo Lucena) possa executar.
- Surge a proposta de pedir aos arquitetos massimos que mostrem um exemplo do material que eles entregam às marcenarias.
- O objetivo seria levar esse exemplo aos marceneiros de interesse para verificar se a informação fornecida é suficiente.
- Considera-se a possibilidade de usar o projeto feito para gui como exemplo, sendo necessário obter a autorização dele para tal.
- @[plaud] - Pedir autorização a gui para compartilhar seu projeto como exemplo.

[05:11-06:27] **Consolidação do plano de ação final para a reunião.**
- toni e carol concordam em solicitar aos arquitetos massimos, já na reunião do dia seguinte, uma demonstração do que é entregue no projeto.
- A intenção é evitar problemas futuros com qualquer marceneiro que venha a ser contratado, não apenas com Lucena.
